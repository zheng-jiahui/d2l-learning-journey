## 🛠️ 环境配置

```bash
# 创建虚拟环境
conda create -n d2l python=3.8
conda activate d2l

# 安装依赖
pip install torch torchvision
pip install matplotlib jupyter d2l
```

或直接安装：
```bash
pip install -r requirements.txt
```
